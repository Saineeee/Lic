# Privacy Policy for Yuri (Discord Bot)

**Last Updated:** March 19, 2026

This Privacy Policy explains how Yuri ("the Bot") collects, uses, stores, and protects your data when you interact with it on Discord. By using the Bot or adding it to your server, you consent to the practices described below.

## 1. What Data We Collect
To function properly as a conversational AI and social bot, Yuri collects the following data:

* **Discord Account Data:** User IDs, Display Names, Avatar URLs, Account Creation Dates, and Server Roles.
* **Presence Data:** Current status (Online, DND) and rich presence activities (e.g., listening to Spotify, playing games), but only at the exact moment a command is run.
* **Message Content:** Text, uploaded images, and audio attachments (voice notes) are processed **only** when the Bot is explicitly mentioned (`@Yuri`) or directly replied to.
* **Command Usage:** Data generated by specific commands, such as target User IDs for `/crush`, `/grudge`, or bug reports via `/feedback`.
* **Server Data:** Guild/Server IDs and specific channel IDs configured by admins for features like `/confess`.

## 2. How We Use Your Data
Your data is used strictly to provide the Bot's core features:

* **AI Inference:** Sending your text, images, or audio to our AI providers to generate Yuri's responses.
* **Conversation Memory:** Maintaining a short-term history of your chat with the Bot so it can remember the context of the conversation.
* **Social Commands:** Analyzing your Discord profile and chat history to power personalized features like `/roast`, `/rate`, and `/ship`.

**We do not use your data to train our own machine learning models, nor do we sell your data.**

## 3. Data Storage and Security
* **Database:** Data is stored securely in a MongoDB database and is encrypted at rest.
* **Automatic Deletion:** Chat history is temporary. All logged conversation data is automatically and permanently deleted from the database after 30 days via a Time-To-Live (TTL) index.

## 4. Third-Party Services
Because Yuri is an AI bot, your inputs (messages, images, audio) are securely transmitted to third-party APIs for processing. These include:

* **Google (Gemini API):** For primary text and image generation.
* **Groq:** For fallback text generation and audio (Whisper) transcription.
* **DuckDuckGo:** For performing web searches based on your queries to provide up-to-date context.

## 5. Data Deletion and User Rights
You have complete control over your conversational data and the right to delete it at any time. 

* **Instant Self-Deletion:** You can instantly and permanently delete your entire chat history with Yuri by using the `/clearhistory` command in any server where the bot is present. This immediately purges your conversational records from our database.
* **Manual Requests:** If you encounter any issues, or if you wish to request the manual deletion of other data (such as bug reports or matchmaking data), you can submit a request directly to the developer using the `/feedback` command.

## 6. Contact Information
If you have any questions or concerns about this Privacy Policy, need assistance with data deletion, or want to report a bug, please use the **`/feedback`** command within any server where Yuri is present. 

This command securely routes your message and User ID directly to the developer's private inbox for review and response.
